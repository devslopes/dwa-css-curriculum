# Text Styling

In the lessons of this section you will learn:

- How text is rendered to the screen
- Properties of the text to be adjusted including:
  - alignment
  - decoration
  - sizing
  - spacing
  - indentation
  - styles (bold, italic, oblique)
  - color
- How to connect external fonts to a website
